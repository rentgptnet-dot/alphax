# Alpha-X Global v2.1

跨市場（台 / 美 / 日）中長期波段決策系統，**新增法人視角過濾**：大盤環境、相對強度 RS、時光機 lookahead bias 修正。

## v2.1 升級重點（vs v2.0）

| 升級項目 | 解決什麼問題 |
|---|---|
| **大盤環境過濾** (Market Regime) | 在熊市裡選股是徒勞 — BEAR 時 L2 自動降為 L1，避免逆勢買進 |
| **相對強度 RS** (Relative Strength) | 法人不買輸大盤的股票 — 90 日跑贏大盤才加分，明顯弱勢扣分 |
| **時光機 lookahead 修正** | yfinance.info 是現值，舊版用 2026 年 EPS 評 2023 年 → 嚴重看多偏誤；現在時光機模式跳過基本面 |
| **回測買進價改用隔日開盤** | 真實世界當天看到收盤訊號時市場已關，舊版用當日收盤是未來函數；現在改用 target_date+1 開盤 |
| **法人級回測指標** | 加上 CAGR / 真實 MDD / 勝率 / 賠率，L2 vs 全樣本對照 |
| **市場溫度計** UI | 掃描結果頂部顯示 BULL / NEUTRAL / BEAR 紅綠燈 |
| **庫存體檢顯示 RS** | 風控儀表板新增 RS_90D 與市場環境欄位 |

## 完整策略堆疊（v2.1）

```
🌐 大盤環境過濾 (200D-MA)        ← 這道沒過，L2 直接降為 L1
   ↓
0️⃣ 流動性過濾 (日均成交 ≥ 5,000 萬 TWD)
   ↓
1️⃣ 趨勢過濾 (Weekly: 30W-MA + 斜率 / Daily: 20D + 60D)
   ↓
1️⃣.5 基本面 (軟性，時光機跳過)
   ↓
1️⃣.6 相對強度 RS (個股 vs 大盤 90 日超額)
   ↓
2️⃣ 確認 (TW: 投信 5 日淨買 ≥ 200 張且正買 3 日 / 美日: 爆量 1.2x)
   ↓
🏷️ 評級 L0 / L1 / L2
```

## 已知限制

1. yfinance 抓基本面 (EPS / 營收成長) 偶爾 None — 已用軟性過濾處理。
2. FinMind 匿名模式有每日呼叫上限，建議申請免費 token。
3. 時光機目前只擬真到「target_date+1 開盤買」，不模擬掛單未成交、滑價、停損掛在當日 high 等。
4. 大盤指數判定用簡單 200D-MA + ±2% 緩衝，未引入 VIX、信用利差等更靈敏的環境指標。
5. RS 只看 90 日，未做產業 RS 排名。
